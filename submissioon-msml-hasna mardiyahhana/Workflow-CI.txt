Deskripsi Workflow:
Workflow CI dibuat menggunakan GitHub Actions dan MLflow Project
untuk mengotomatisasi proses training dan evaluasi model machine learning.

Workflow dijalankan secara otomatis ketika terjadi push atau pull request
pada repository.

Tahapan Workflow CI:
1. Checkout repository
2. Setup environment berdasarkan conda.yaml
3. Menjalankan MLflow Project
4. Logging eksperimen dan artifact ke MLflow Tracking

Artefak yang dihasilkan:
- Model machine learning
- Metrik evaluasi (accuracy, precision, recall, f1-score)
- Artifact MLflow (model dan environment)

Kesimpulan:
Workflow CI berhasil mengotomatisasi proses training model machine learning
menggunakan MLflow Project sesuai dengan kriteria submission Dicoding.
